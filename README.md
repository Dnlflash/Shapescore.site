# Shapescore

Site que calcula seu potencial estetico e em seguida mostra pagina de pagamento para mais resultados como a gordura corporal usando o calculo da marinha americana

usa:
html,css,js e python com o flask

baixe os arquivos e execute o python, copie a url do terminal e abra no navegador
ou digite shapescore.vercel.app para usar
